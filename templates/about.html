🎨 About This Project

Welcome to the Art Gallery Management System!

This project was built to help manage artists, artworks, collectors, and sales efficiently. It allows users to add new artworks, register collectors, track artwork history, and generate useful reports.

Built with ❤️ using FastAPI, SQLAlchemy, Jinja2, and Oracle Database, this system aims to provide a smooth and professional experience for managing gallery operations.
